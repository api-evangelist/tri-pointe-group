# Tri Pointe Homes

Profile for **Tri Pointe Homes, Inc.** (NYSE: **TPH**) in the API Evangelist network. Fortune 1000 homebuilder (rank ~766).

- Website: https://www.tripointehomes.com
- Investor Relations: https://investors.tripointehomes.com
- Headquartered in Orange County, California; founded 2009
- Operates in 13 states + DC under a unified Tri Pointe Homes brand (consolidated 2021), with legacy regional divisions Pardee Homes, Maracay, Quadrant Homes, Winchester Homes, and Trendmaker Homes
- Ancillary services: Tri Pointe Connect (mortgage), Tri Pointe Assurance (title & escrow), Tri Pointe Advantage (insurance), LivingSmart (smart-home package)
- Scale snapshot (LTM as of 2026-03-31): 161 active communities, 4,643 deliveries, $3.15B revenue, $678K average sales price; ~58,000 homes built since 2009

## API surface

**Tri Pointe Homes has no public developer program, no documented API, no OpenAPI/Swagger/Postman artifacts, no SDKs, no developer portal, no status page, and no GitHub organization.** Its digital footprint is limited to a marketing website, investor portal, blog, careers site, and social channels — all enumerated in `apis.yml`.

This repo is therefore a **provider profile only** — no `openapi/`, `capabilities/`, `rules/`, or related artifact folders are present, in accordance with the API Evangelist pipeline rule against creating empty/placeholder specs.
